Componentes UI reutilizables para toda la app.

Primitivas base disponibles:

- `Card`, `CompactCard`, `SidePanel`
- `Button`, `IconButton`
- `FieldShell`, `Input`, `Select`, `Textarea`
- `Badge`, `StatusBadge`
- `EmptyState`
- `MetricGrid`

Todos deben consumir los tokens definidos en `app/globals.css`.
